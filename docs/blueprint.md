# **App Name**: Gordo Salgado 

## Core Features:

- Responsive Design: Fully responsive design to ensure a seamless experience on all devices.
- Menu Display: Display menu items in a grid layout, including images, descriptions, and prices.
- Category Filtering: Enable users to filter menu items by categories such as 'Fritos', 'Assados', etc.
- Order via WhatsApp: Allow users to place orders by redirecting them to WhatsApp with a pre-filled message.
- Location Map: Display the store location using Google Maps.

## Style Guidelines:

- Primary color: Gold-Yellow (#FFC107) to stimulate appetite and evoke the feeling of warm, fried snacks. 
- Accent color: Tomato-Red (#D32F2F) as a classic food color to create urgency and call to action.
- Background color: Desaturated yellow (#FAF7E7). Dark gray (#212121) for legible text.
- Headline font: 'Belleza' (sans-serif) for headlines, providing personality suited to food, art, and design; body text: 'Alegreya' (serif) to ensure legibility in menu descriptions and other longer text sections.
- Implement a 'bento grid' layout with rounded borders. Utilize soft shadows to add depth to photos.
- Use custom icons related to food, delivery, and trust signals.
- Apply subtle fade-in animations when loading sections and transitions for interactive elements.
